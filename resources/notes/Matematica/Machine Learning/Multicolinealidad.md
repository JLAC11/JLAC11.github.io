# Multicolinealidad

Sea $X=[X^{(1)},...,X^{(n)}]$ un vector de predictores de un [modelo](../../Modelo.md). Entonces, si la [correlaci√≥n](Matriz%20de%20covarianza.md) $\mathrm{Corr}(X^{(i)},X^{(j)})\approx1$ para algunos $i$ y $j$ arbitrarios, entonces existe multicolinealidad.

Pueden observarse los *eigenvalores* de la matriz, y si existe mucha diferencia en magnitud entre ellos, entonces hay un problema de colinealidad.
